# Hospital Patient Management System

## Overview
This project is a patient management system developed using tree data structures to efficiently store and manage patient records.

## Technologies Used
- Programming Language: (Python)
- Data Structures: Trees

## Features
- Add new patient records
- Search patient data efficiently
- Organized data storage using tree structure
- Fast retrieval of patient information

## How to Run
1. Install python language
2. Run the main file

## Concept Used
- Tree Data Structure for efficient data management
